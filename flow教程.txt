js文件头部需要加上这个标志
//@flow或者/* @flow */
var foo:?string=null
类型有
string
number
object
array
全部小写


https://flow.org/en/docs/types/functions/
function truthy(a, b): boolean %checks {
  return (!!(a && b)) ;
}
加上这个%checks会真实的校验
